# Movie API

This is a simple Node.js application that provides a RESTful API for managing a collection of movies. The movies are stored in a JSON file (`movies.json`), and the API supports basic CRUD operations.

## Features

- Create a new movie
- Read all movies
- Read a single movie by index
- Update a movie by index
- Delete a movie by index
